# Astro + Keystatic (Vercel)

## Quick Start
1. Add files to your repo root.
2. In Vercel → Project → Environment Variables, set:
   - KEYSTATIC_GITHUB_CLIENT_ID
   - KEYSTATIC_GITHUB_CLIENT_SECRET
   - KEYSTATIC_GITHUB_REPO = owner/repo
   - KEYSTATIC_GITHUB_BRANCH = main
3. Redeploy on Vercel.
4. Open `/keystatic` to sign in and edit content.

## Local
npm i
npm run dev
# http://localhost:4321/keystatic
